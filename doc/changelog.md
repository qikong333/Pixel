# Changelog

## 2026-04-23

- 新增 `Pyxelate` 作为第 5 种像素化算法。
- 首页对比结果新增 `Pyxelate` 列及对应 SVG 下载链接。
- 新增 `test_generate.py`，覆盖 `pyxelate` 输出与页面集成。
- 修正 `server.py` 中 `num_colors` 的默认值为 `5`。

## 2026-05-07

- 首页改为单张图片上传模式，移除了默认样例图和默认参数值。
- 新增上传后自动读取原图宽高并填入推荐参数的前端逻辑。
- 服务端改为 `POST /generate` 处理图片上传与同图重复调参。
- 生成流程改为纯内存一次性响应，不再写入上传图和结果文件。
- 新增生成中的 loading 遮罩。
- 结果页改为只内嵌缩略图，默认关闭 SVG。
- 新增 `/media/...` 内存路由，点击缩略图时再按需加载大图。
- 新增 `/clear-session` 清理路由，刷新页面时会主动清空当前上传图和生成结果的内存数据。
- 每个算法结果卡片新增 `下载 JPG` 按钮，可直接下载当前生成的大图。
