# Testing

## Automated

- `.venv/bin/python -m unittest test_generate.py`

验证内容：
- `method_pyxelate()` 能生成像素化结果。
- 首页空状态包含上传表单，且没有默认图片和默认数值。
- 单图生成链路会回填当前参数值。
- 结果页只内嵌缩略图，不再内嵌 SVG，也不再依赖 `uploads/` / `output/` 文件路径。
- 服务端媒体会话会为大图提供按需加载地址，并能清理过期资源。
- 页面刷新或离开时会通过 `/clear-session` 主动清理当前会话。

## Manual

- `.venv/bin/python server.py`
- 打开 `http://localhost:8082/`，确认初始页没有默认图片和默认参数值。
- 上传一张图片，确认页面自动填入原图宽高、像素格 `1x1`、分色 `5`。
- 生成后确认结果页包含 5 个算法卡片，并可再次调参重生成。
- 生成时确认页面显示 loading 遮罩。
- 确认结果页源码中只有缩略图 `data:image/jpeg;base64,...`，大图地址为 `/media/...`。
- 点击缩略图后确认浏览器能正常加载对应大图。
- 点击 `下载 JPG` 后确认能下载对应算法的大图结果。
- 刷新页面后确认结果被清空，需要重新上传图片，同时之前的 `/media/...` 地址失效。
